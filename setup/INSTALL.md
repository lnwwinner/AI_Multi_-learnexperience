# Installation Guide

## 1. Clone Repo
```
git clone https://github.com/lnwwinner/AI_Multi_-learnexperience.git
cd AI_Multi_-learnexperience
```

## 2. Install Requirements
```
pip install -r requirements.txt
pip install flask
pip install iqoptionapi
```

## 3. Setup Environment
Create .env file:
```
IQ_EMAIL=your_email
IQ_PASSWORD=your_password
```

## 4. Run System
```
python main.py
```

## 5. Run Dashboard
```
python dashboard/web_dashboard.py
```
Open browser: http://localhost:5000

---

## Notes
- Start with PRACTICE account only
- Set low amount for testing
- Monitor dashboard before real trading
